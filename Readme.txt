The scripts in this folder is used to format the input values. Input variables can be found from Input_List.csv in this folder

The script includes:
1 - Get the background values (e.g., SOC), parameter values (e.g., EF_NOx), NP input (e.g., manure N input) at global scale
2 - Cut the required input for study areas

All of the processed data generated using the scripts in this folder can be found from https://doi.org/10.5281/zenodo.15351242
